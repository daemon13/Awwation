Awwation

Create Zooming Prezi-like HTML5 presentations!

Navigate with ← and → keys.

See it in Fullscreen! Or, try the Editor.

Awwation is a presentation creator app, inspired by Prezi.

It runs entirely in the browser, purely client-side, and generates SVG presentations. For now, it works properly only with Google Chrome. Presentation files are created client-side using an implementation of the new FileSaver API, with no server-side crunching required.

The editor is built on top of the core of SVG-edit.

Once the presentation sequence is defined in the editor, it generates an array of SVG element ID's belonging to the path nodes, so you can easily take that sequence and pass it to an animation backend like Sozi, dizzy.js, or impress.js.
Currently, only the Sozi backend is implemented.
