Awwation by adityab

Prezi clone in HTML5, Js, and SVG.

Awwation is a presentation creator inspired by Prezi. It runs entirely in the browser, purely client-side, and generates SVG presentations. Presentation files are created client-side using an implementation of the new FileSaver API, with no server-side crunching required. Once the presentation sequence is defined in the editor, it generates an array of SVG element ID's belonging to the path nodes, so you can easily take that sequence and pass it to an animation backend like Sozi, dizzy.js, or impress.js. Currently, only the Sozi backend is implemented. As of right now, the editor is a very badly hacked modification of SVG-edit, and there may be minor glitches in drawing or the UI. This will be fixed soon - I have coursework to finish!
